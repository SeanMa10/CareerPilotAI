# CareerPilot AI

CareerPilot AI is a full-stack student portfolio project designed to help users improve their career readiness through AI-powered tools such as resume review, skill gap analysis, roadmap generation, and interview practice.

## Project Purpose

This project was built as a student portfolio and educational showcase project.
It demonstrates product thinking, full-stack development, authentication, file uploads, MongoDB integration, AI-powered workflows, and responsive UI design.

## Main Features

- User authentication with JWT
- Profile creation and editing
- Resume PDF upload and text extraction
- AI Resume Review
- AI Skill Gap Analysis
- AI Roadmap Generator
- AI Interview Coach
- PDF export for AI reports
- Free usage gating for premium AI features
- Legal pages: Terms of Use and Privacy Policy
- Responsive landing page and dashboard

## Tech Stack
